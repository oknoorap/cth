# CTH (CSV to Static HTML)
> CTH is a command-line tool to converts csv file to HTML.

## Install
```bash
# NPM
npm install cth -g

# Yarn
yarn global add cth
```

## Usage
- Please type `--help` in your terminal.
- Place your csv data in csv folder.
